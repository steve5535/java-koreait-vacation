# git?
- 파일 버전 관리 툴

## git 명령어
1. 로컬환경 셋팅
- 내 프로젝트에 git 적용: git init
- 작업자 등록
- git config user.name 내아이디
- git config user.email 내이메일

2. 변경사항 저장
- 임시공간에 save: git add 파일명
- 새로생긴 모든 파일 임시 공간 save: git add .
- 임시공간에 있는 모든save들을 저장: git commit -m "조건문"

3. 원격저장소(github)와 연결
- 프로젝트마다 최초 1회만 수행
- git remote add origin 원격레포지토리주소

4. 원격저장소(github)에 업로드
- commit들을 원격저장소에 저장: git push
- 최초에 push시, git push -u origin main
